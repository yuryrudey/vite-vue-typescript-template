## Vite + Vue + TypeScript + ESLint + Stylelint + Prettier

First of all you need to install the dependencies.

```
yarn
```

### Commands

Starts development server.

```
yarn dev
```

Builds for production.

```
yarn build
```

Starts test server for production build.  
**NOTE**: before that you need to execute the `yarn build` command.

```
yarn preview
```

Lints script files.  
**NOTE**: to fix what can be fixed automatically, add `--fix` flag.

```
yarn lint:script
```

Lints style files.  
**NOTE**: to fix what can be fixed automatically, add `--fix` flag.

```
yarn lint:style
```

Formats all files.

```
yarn format
```
