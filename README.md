## Vite + Vue + TypeScript + Vitest + ESLint + Stylelint + Prettier

First of all you need to install the dependencies.

```
pnpm i
```

### Commands

Start development server.

```
pnpm dev
```

Build for production.

```
pnpm build
```

Start test server for production build.  
**NOTE**: before that you need to execute the `pnpm build` command.

```
pnpm preview
```

Lint script files.

```
pnpm lint:script
```

Lint and fix script files.

```
pnpm lintfix:script
```

Lint style files.

```
pnpm lint:style
```

Lint and fix style files.

```
pnpm lintfix:style
```

Lint all files.

```
pnpm lint
```

Lint and fix all files.

```
pnpm lintfix
```

Format all files.

```
pnpm format
```

Run tests.

```
pnpm test
```
